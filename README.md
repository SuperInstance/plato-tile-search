# plato-tile-search

Nearest-neighbor tile search with keyword overlap, domain matching, and composite ranking

Part of the [PLATO framework](https://github.com/SuperInstance) — 72 crates for deterministic AI knowledge management.

## Tile Pipeline

validate → score → dedup → store → search → rank → prompt

## Usage

```toml
[dependencies]
plato-tile-search = "0.1"
```

Zero external dependencies. Compatible with cargo 1.75+.
